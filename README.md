# tiktokio

TikTok Video Downloader Website - A PHP-based web application for downloading TikTok videos without watermarks.

## Features

- Download TikTok videos (with/without watermark)
- Download TikTok videos as MP3
- Download TikTok stories
- Multi-language support
- Admin panel for content management
- SEO optimized pages
- Download tracking and analytics

## Requirements

- PHP 8.2+
- MySQL/MariaDB 9.5+
- Web server (Apache/Nginx) or PHP built-in server

## Installation

1. Clone the repository
2. Import the database: `tiktokio_mobi.sql`
3. Configure database connection in `includes/config.php`
4. Start the PHP server: `php -S localhost:8000 php_router.php`
